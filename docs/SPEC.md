# SPEC — 말싸움 연구소 (MSW Lab) 기능 명세

## 페이지 구조

```
/ (홈 = 상황 설정)
/chat (말싸움 채팅)
/result (결과 리포트)
```

---

## 페이지 1: 상황 설정 (`/`)

### 화면 구성
- 서비스 로고 + 한줄 소개
- 입력 폼:
  - [대상] 텍스트 입력 (placeholder: "예: 직장 상사, 남자친구, 친구")
  - [상황] 텍스트에어리어 (placeholder: "어떤 상황인지 설명해주세요. 예: 친구가 약속을 또 어겼는데 자기 잘못이 아니라고 우김")
- [말싸움 시작하기] 버튼

### 동작
1. 사용자가 대상 + 상황 입력 후 버튼 클릭
2. 입력값 유효성 검사 (둘 다 비어있으면 버튼 비활성화)
3. 입력값을 sessionStorage 또는 zustand state에 저장
4. `/chat` 페이지로 이동

### AI 프롬프트 설계 (시스템 프롬프트)
```
당신은 말싸움 연습 상대입니다.
대상: {대상}
상황: {상황}

규칙:
- 해당 대상의 입장에서 현실감 있게 반응하세요
- 너무 쉽게 지지 말고, 적당히 반박하고 억울함을 표현하세요
- 욕설·혐오표현은 사용하지 마세요
- 대화가 자연스럽게 마무리되는 시점(5~10턴 이후 흐름이 정리될 때)에 
  "이 정도면 충분히 연습한 것 같아요. 결과를 볼까요?" 라고 제안하세요
- 한국어로 답하세요
- 답변은 2~4문장으로 짧고 현실적으로
```

## 페이지 2: 말싸움 채팅 (`/chat`)

### 화면 구성
- 상단: 상황 요약 텍스트 (접을 수 있는 토글)
- 채팅 영역:
  - AI 메시지: 왼쪽 말풍선 (회색)
  - 사용자 메시지: 오른쪽 말풍선 (포인트 컬러)
  - AI 타이핑 중: 점 3개 애니메이션
- 하단 입력창:
  - 텍스트 입력 필드
  - 전송 버튼
  - [여기서 끝내기] 버튼 (항상 표시)

### 동작
1. 페이지 진입 시 AI가 먼저 선제 발언 (상황에 맞게 시작)
2. 사용자 입력 → API 호출 → AI 응답 스트리밍
3. AI가 종료 제안 시:
   - 모달 팝업: "결과를 확인할까요?" [확인] [계속하기]
4. [여기서 끝내기] 클릭 시:
   - 모달 팝업: "지금까지의 대화로 결과를 분석할게요" [확인] [취소]
5. 종료 확정 시 대화 내역 저장 후 `/result`로 이동

### 데이터 관리
- 대화 내역: `messages[]` 배열로 state 관리
- 형태: `{ role: 'user' | 'assistant', content: string, timestamp: Date }`
- 페이지 이동 시 결과 분석용으로 전달

## 페이지 3: 결과 리포트 (`/result`)

### 화면 구성

#### 섹션 A: 종합 점수
- 원형 게이지 또는 큰 숫자로 점수 표시 (0~100)
- 점수 등급 라벨 (예: "말싸움 중수", "감정 폭발형", "논리 마스터")

#### 섹션 B: 항목별 분석
분석 항목 4가지:
1. **논리력** — 주장의 일관성과 근거
2. **감정 조절** — 흥분하지 않고 침착하게 대응했는지
3. **표현력** — 하고 싶은 말을 명확하게 전달했는지
4. **설득력** — 상대방의 입장을 변화시킬 수 있었는지

각 항목: 점수(0~100) + 2~3줄 설명

#### 섹션 C: 피드백
- **잘한 점** (최소 2개)
- **개선할 점 + 다음에 이렇게 해보세요** (구체적 예시 포함, 최소 2개)

#### 섹션 D: 공유 이미지 다운로드
- 미리보기 카드 (이름 없이 점수 + 등급 + 한줄 평)
- [이미지 저장하기] 버튼 → html2canvas로 PNG 다운로드
- [다시 하기] 버튼 → `/`로 이동

### AI 분석 프롬프트
```
아래 말싸움 대화를 분석해주세요.

대상: {대상}
상황: {상황}
대화내역: {messages}

다음 JSON 형식으로만 응답하세요:
{
  "totalScore": 75,
  "grade": "말싸움 중수",
  "categories": [
    { "name": "논리력", "score": 80, "description": "..." },
    { "name": "감정 조절", "score": 70, "description": "..." },
    { "name": "표현력", "score": 75, "description": "..." },
    { "name": "설득력", "score": 65, "description": "..." }
  ],
  "strengths": ["...", "..."],
  "improvements": [
    { "point": "...", "tip": "다음에는 이렇게 해보세요: ..." },
    { "point": "...", "tip": "다음에는 이렇게 해보세요: ..." }
  ],
  "oneLineSummary": "감정은 잘 잡았지만 논리로 밀어붙이는 힘이 아쉬웠어요"
}
```

### 공유 이미지 스펙
- 크기: 1080x1080px (정사각형, 인스타그램 최적화)
- 내용: 로고 + 점수 + 등급 + 한줄 요약 + "msw-lab.com"
- 배경: 브랜드 컬러 그라데이션

## API 연동

### 사용 API
- **Anthropic Claude API** (claude-sonnet-4-6 모델)
- Next.js API Route로 키 보호 (`/api/chat`, `/api/analyze`)

### `/api/chat`
```
POST /api/chat
Body: { messages: Message[], situation: string, target: string }
Response: 스트리밍 텍스트
```

### `/api/analyze`
```
POST /api/analyze
Body: { messages: Message[], situation: string, target: string }
Response: JSON (분석 결과)
```
