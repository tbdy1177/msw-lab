# POLICY — 말싸움 연구소 서비스 정책

## 등급 체계

점수는 AI가 산출하고, **등급 라벨은 프론트에서 점수 기준으로 고정 적용**한다.
(AI가 반환하는 `grade` 필드는 무시)

| 점수 범위 | 등급 |
|-----------|------|
| 90 ~ 100 | 말싸움 연구소장 |
| 80 ~ 89 | 수석 연구원 |
| 70 ~ 79 | 책임 연구원 |
| 60 ~ 69 | 연구원 |
| 50 ~ 59 | 연구 인턴 |
| 40 ~ 49 | 견습생 |
| 0 ~ 39 | 관찰 대상자 |

> 컨셉: "말싸움 연구소" 세계관에서 점수가 높을수록 연구자 직급이 높아지고,
> 낮으면 연구 대상이 된다는 반전 유머를 담음.

## 결과 분석 로딩 메시지

분석 대기 중 아래 5개 문구를 **순서대로 3초 간격으로 롤링** 표시.
전환 시 0.4초 페이드 아웃→인 적용.

1. 대화를 분석하고 있어요...
2. 조금만 기다려주세요...
3. 많이 화났죠?
4. 그동안 당신이 말싸움에 실패했던 원인...
5. 정확하게 분석해 드립니다...

---

## 채팅 종료 조건

- **사용자 수동 종료**: [여기서 끝내기] 버튼 → 확인 모달 → `/result`
- **AI 종료 제안**: AI 응답에 "결과를 볼까요?" 포함 시 → 확인 모달 자동 노출
- 최소 대화 수: 2턴 미만 시 종료 불가 (토스트 안내)

---

## AI 프롬프트 정책

### 채팅 AI (`/api/chat`)
- 말싸움 상대 역할: 상황에 맞게 억울함·반박 표현
- 욕설·혐오 표현 금지
- 답변 길이: 2~4문장
- 5~10턴 이후 흐름이 정리되면 종료 제안

### 분석 AI (`/api/analyze`)
- 4개 항목 평가: 논리력, 감정 조절, 표현력, 설득력
- 잘한 점 최소 2개, 개선할 점 최소 2개 (구체적 팁 포함)
- `grade` 필드는 프론트에서 덮어쓰므로 참고용으로만 생성

## 데이터 보관 정책

- MVP 단계: 로그인 없음, 서버에 데이터 저장하지 않음
- 모든 상태는 Zustand store (클라이언트 메모리)에만 보관
- 페이지 새로고침 시 초기화
