# PRD — 말싸움 연구소 (MSW Lab)

## 1. 서비스 개요

| 항목 | 내용 |
|------|------|
| 서비스명 | 말싸움 연구소 (MSW Lab) |
| 한줄 소개 | 말싸움 연습부터 면접·회의 시뮬레이션까지, AI와 함께 말하는 힘을 키우는 서비스 |
| 타겟 유저 | 말싸움에서 항상 지고 나중에 후회하는 사람, 자기표현이 어려운 사람 |
| 핵심 가치 | "뒤돌아서 후회하지 말자" — 미리 연습하고, 분석받고, 성장한다 |

---

## 2. 문제 정의

- 말싸움·논쟁에서 하고 싶은 말을 못 하고 뒤늦게 후회하는 사람이 많다
- 자기표현 능력은 연습으로 향상되지만, 현실에서 반복 연습할 기회가 없다
- 기존 AI 챗봇은 말싸움 상대 역할을 제대로 수행하지 않는다

---

## 3. 솔루션

AI가 실제 말싸움 상대 역할을 하고, 연습 후 객관적인 피드백과 점수를 제공한다.
결과를 이미지로 다운로드해 SNS에 공유할 수 있어 바이럴 확산을 유도한다.

---

## 4. 로드맵

### Phase 1 — MVP (현재 목표)
- 말싸움 시뮬레이션
- 점수 및 상세 분석 리포트
- SNS 공유용 결과 이미지 다운로드

### Phase 2 — 고도화
- 면접 준비 시뮬레이션
- 회의·협상 시뮬레이션
- 회원가입 / 로그인 (기록 저장, 성장 히스토리)
- AI 캐릭터 선택 기능 (성격 유형별)
- 난이도 설정

### Phase 3 — 확장
- 유료 플랜 (상세 리포트, 무제한 연습)
- 리더보드 / 커뮤니티

---

## 5. 핵심 기능 (MVP)

### 5-1. 상황 설정 (페이지 1)
- 사용자가 말싸움 대상을 입력 (예: 친구, 직장동료, 가족)
- 상황 설명을 자유 텍스트로 입력
- AI가 상황을 분석해 적절한 말싸움 상대 캐릭터로 자동 설정

### 5-2. 말싸움 채팅 (페이지 2)
- 실시간 채팅 기반 말싸움 진행
- AI는 상황에 맞는 톤·스타일로 반응 (공격적, 억울한 척, 논리적 반박 등)
- 수동 종료 버튼 항상 표시
- AI가 대화 흐름이 마무리될 때 "여기서 끝낼까요?" 제안

### 5-3. 결과 분석 리포트 (페이지 3)
- 100점 만점 종합 점수
- 항목별 분석 (논리력, 감정조절, 표현력, 설득력 등)
- 잘한 점 (장점)
- 개선할 점 + 구체적인 다음번 활용 팁
- SNS 공유용 결과 이미지 다운로드 버튼

---

## 6. 비기능 요구사항

- 모바일 우선 반응형 디자인
- 로그인 없이 바로 사용 가능 (MVP)
- 로딩 중 스켈레톤 UI 또는 타이핑 인디케이터 표시
- 결과 이미지는 클라이언트사이드에서 생성 (html2canvas 또는 canvas API)

---

## 7. 성공 지표 (MVP)

- 서비스 완주율 (설정 → 채팅 → 결과까지 이탈 없이 완료한 비율) > 60%
- 결과 이미지 다운로드율 > 30%
- 재방문율 (같은 사용자가 2회 이상 사용) > 20%
